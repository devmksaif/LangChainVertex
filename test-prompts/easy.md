# Easy Workflow Prompts

## Prompt 1: Simple Email Notification
Create a workflow that sends a welcome email to a new user when a webhook is triggered with their email address.

## Prompt 2: Basic Log Action
Build a workflow that logs a message "Hello World" when started manually.

## Prompt 3: REST API Call
Design a workflow that makes a GET request to https://jsonplaceholder.typicode.com/posts/1 and logs the response.
